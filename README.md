# DEXEL-IP-INTEL
DEXEL IP INTEL is a Python-based terminal tool for advanced IP address intelligence and risk analysis. It provides geolocation data, ISP and organization details, reverse DNS, WHOIS lookup, common port scanning, subnet info, and risk scoring with TXT/JSON export support. #Python #CyberSecurity #IPIntel #Networking #EthicalHacking #SecurityTool
